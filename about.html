About

This site holds a collection of Surinamese radio stations and some radio stations airing mostly late 90s & early 2000s Bollywood songs. It's intended for private/non-commercial use.

Legality

The streams are not hosted and/or modified by me.
The stream URLs are gathered from the web and were publicly accessible. If you have any comment or want to remove/add your/a station, use the contact information on the top of this page.

Http vs. Https

A valid SSL certificate is present on the site but is not used by default. That's because some of the radio stations stream their content over http. And it's not possible to access http resources over https.

Https is more secure, but because this is a static website (no server-side coding involved) http (most likely) won't cause any risk. TL;DR: If you want to be able to listen to all the stations, use http. If you want to be on the safer side, use https (not all stations available) !

Code

The site relies heavily on JavaScript (>ES 6).
  • URLs are gathered from the web
  • These are stored in a JSON object
  • Object is looped through to create station blocks
  • Functionalities (e.g: play, pause, time update) are added

Media Controls

The following media controls are added and can be used on browsers that support the Media Session API (e.g: Chromium based browsers):
  • Jump to previous station
  • Jump to next station
These can be used to switch between stations in the background, without opening the browser/page.
